# 🎵 Valeon API

Valeon est une application intelligente permettant d'identifier automatiquement les musiques, films, images et vidéos. Cette API fournit tous les endpoints nécessaires pour le backend.

## ✨ Fonctionnalités

### Core
- 🔍 Scan audio avec reconnaissance musicale (AudD, Whisper)
- 🖼️ Scan image avec analyse GPT-4 Vision
- 🎬 Scan vidéo avec extraction de frames
- 📱 Support de multiples sources (fichier, microphone, caméra, galerie)

### Reconnaissance
- 🎵 Musique (Spotify, AudD)
- 🎬 Films et séries (TMDB)
- 🖼️ Images et objets
- 📚 Livres et produits

### Bibliothèque utilisateur
- ⭐ Favoris
- 📋 Playlists personnalisées
- 📊 Historique des scans
- 📈 Statistiques d'utilisation

### Recommandations IA
- 🤖 Recommandations personnalisées
- 💬 Chat IA conversationnel
- 📊 Algorithmes collaboratifs et basés sur le contenu
